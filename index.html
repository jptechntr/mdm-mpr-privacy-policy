Privacy Policy

MDM MPR is a utility application designed to assist users in calculation and report preparation related to the PM POSHAN (Mid-Day Meal) program.

Data Collection and Storage

This application does not collect, store, or transmit any personal information to external servers. All data entered by the user remains stored locally on the user’s device.

Internet Usage

The application uses internet connectivity only to open official government website links in the user’s default browser for informational updates. The app itself does not transmit user data over the internet.

Third-Party Services

This application does not use analytics services, advertisements, tracking tools, or third-party data collection services.

Affiliation Disclaimer

This application is an independent utility tool and is not affiliated with, endorsed by, or officially connected to the Government of India or PM POSHAN authorities.
